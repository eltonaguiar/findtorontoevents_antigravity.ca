üöÄ Quick Start: Forward Test in 30 Minutes

Follow these exact steps to get your first forward test running today. No programming experience required.

1

Create Alpaca Paper Account

Go to Alpaca, sign up for a paper trading account. It's free, no credit card needed. Copy your API Key ID and Secret Key.

Paper API Endpoint: paper-api.alpaca.markets WebSocket: wss://stream.data.alpaca.markets/v2/iex
2

Add Webhook URL to TradingView

Set up a webhook receiver (use webhook.site for testing) or connect to your bot. Copy the webhook URL.

Free webhook tester: https://webhook.site/ (You'll get a unique URL like: https://webhook.site/abc-123-xyz)
3

Configure Pine Script Alerts

Add this to your Pine Script strategy to send alerts to your webhook when signals trigger.

// Add to your Pine Script: alertcondition(buySignal, "Buy Signal", "BUY: {{ticker}} @ {{close}}") alertcondition(sellSignal, "Sell Signal", "SELL: {{ticker}} @ {{close}}") // In TradingView alert message: {"action": "buy", "ticker": "{{ticker}}", "price": {{close}}, "time": "{{time}}"}
4

Create Simple Python Bot

This basic bot receives webhook alerts and places paper trades via Alpaca.

from flask import Flask, request import alpaca_trade_api as tradeapi API_KEY = 'YOUR_ALPACA_KEY' API_SECRET = 'YOUR_ALPACA_SECRET' BASE_URL = 'https://paper-api.alpaca.markets' api = tradeapi.REST(API_KEY, API_SECRET, BASE_URL) app = Flask(__name__) @app.route('/webhook', methods=['POST']) def webhook(): data = request.json if data['action'] == 'buy': api.submit_order( symbol=data['ticker'], qty=10, side='buy', type='market', time_in_force='day' ) return 'OK', 200 if __name__ == '__main__': app.run(port=5000)
5

Deploy and Monitor

Deploy your bot (use ngrok for local testing, or PythonAnywhere for free hosting). Watch your paper trades execute in real-time!

# Install dependencies: pip install flask alpaca-trade-api # Run locally with tunnel: ngrok http 5000 # Then set TradingView webhook to your ngrok URL

üí° Pro Tips for Forward Testing

üìà Test Multiple Timeframes

Run the same strategy on 1min, 5min, and 15min charts simultaneously. Different timeframes can produce very different results.

üìä Track Everything

Log every trade, signal, and market condition. Use Google Sheets (free) to track: Entry price, exit price, slippage, fees, time held.

üéØ Set Minimum Test Period

Forward test for at least 30 days or 100 trades before trusting results. One good week doesn't mean the strategy works.

üí∞ Include Slippage

Add 0.1% slippage to your paper trades. Real execution is always worse than theoretical. Account for spread on crypto.

‚è∞ Test Different Sessions

Markets behave differently at market open (9:30), lunch (12:00), and close (16:00). Test all sessions separately.

üîß Start Simple

Begin with 1 strategy, 1 symbol, small size. Scale up complexity only after consistent profitability.
