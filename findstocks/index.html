Honest Status Report: Paused Stock Algorithms

7 stock-specific algorithms are currently paused due to a critical disconnect between signal quality and execution:

  • Cursor Genius — 85.71% directional accuracy but 11.54% execution win rate (tight stops killed good picks)
  • ETF Masters — 82.35% directional accuracy but 3.37% execution (ETF daily swings hit 2% stop losses)
  • Blue Chip Growth — 80.00% directional accuracy but 5.56% execution (dividend gaps triggered stops)
  • Sector Rotation — 72.73% directional accuracy but 2.19% execution (sector themes need longer hold periods)
  • Technical Momentum — 25% directional accuracy, 0% execution (genuinely poor signals, filters relaxed Feb 12)
  • Composite Rating — 50% directional accuracy, 0% execution (too restrictive filters, relaxed Feb 12)
  • Sector Momentum — 100% direction on only 4 picks (statistically meaningless sample)

Fix deployed Feb 12: EGARCH volatility-adjusted stops, meta-label XGBoost filtering, portfolio risk controller, and data quality monitor. Re-enablement expected within 1-2 weeks after validation.

Key insight: The algorithms pick direction correctly 80%+ of the time. The problem was stop losses were too tight for actual market volatility. EGARCH dynamically sizes stops to each asset's real volatility.
