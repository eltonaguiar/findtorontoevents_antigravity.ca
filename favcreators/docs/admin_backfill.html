Backfill Guest List to Existing Users

This tool will copy the guest list (user_id=0) to all existing users who don't have a creator list yet.

This ensures that:

  • All users start with the same default creators
  • Users can then customize their list (add/remove creators)
  • Changes are tracked in the user_lists table

Note: Users who already have a list will be skipped.
