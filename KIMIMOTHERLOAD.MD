```python
    # Views: Our predictions
    P, Q = encode_views(views)  # P = pick matrix, Q = return forecasts
    
    # Omega: Confidence in views (uncertainty)
    omega = view_confidences
    
    # Posterior: Combine prior + views (Bayesian)
    expected_returns = posterior_returns(pi, P, Q, omega)
    
    # Optimize with new expected returns
    return mean_variance_optimization(expected_returns, covariance)
```

```python
def delta_hedge(portfolio_options, underlying_positions):
    """
    Neutralize directional exposure

    total_delta = sum([opt['delta'] * opt['quantity'] for opt in portfolio_options])
    total_delta += sum([pos['delta'] for pos in underlying_positions])
```

```python
    Use microstructure to time entries/exits
```

## APPENDIX: KEY METRICS DEFINITIONS

### Return Metrics
- **Total Return:** (Final Value / Initial Value) - 1
- **Annualized Return:** (1 + Total Return)^(365/Days) - 1
- **Alpha:** Excess return vs benchmark
- **Information Ratio:** Alpha / Tracking Error

### Risk Metrics
- **Sharpe Ratio:** (Return - Risk Free) / Volatility
- **Sortino Ratio:** (Return - Risk Free) / Downside Volatility
- **Calmar Ratio:** Annual Return / Max Drawdown
- **Max Drawdown:** Peak to trough decline

### Trade Metrics
- **Win Rate:** Winning Trades / Total Trades
- **Profit Factor:** Gross Profit / Gross Loss
- **Expectancy:** (Win% × Avg Win) - (Loss% × Avg Loss)
- **Payoff Ratio:** Avg Win / Avg Loss

### Statistical Significance
- **Deflated Sharpe Ratio:** Adjusts for multiple testing
- **T-Statistic:** Significance of returns
- **P-Value:** Probability results are random
